# SuiLight 知识沙龙 - Streamlit 版本

这个版本可以直接部署到 Streamlit Cloud，无需服务器！

## 一键部署

[![Streamlit App](https://static.streamlit.io/badges/streamlit_badge.svg)](https://share.streamlit.io/wanyview/suilight/main/ui/streamlit_app.py)

## 部署步骤

### 1. 访问 Streamlit Cloud
访问 https://share.streamlit.io

### 2. 连接 GitHub
- 点击 "New app"
- 选择 "Connect to GitHub"

### 3. 选择仓库
- Repository: `wanyview/SuiLight`
- Branch: `main`
- Main file path: `ui/streamlit_app.py`

### 4. 部署
- 点击 "Deploy!"

## 本地运行

```bash
pip install streamlit
streamlit run ui/streamlit_app.py
```

## 访问
- 本地: http://localhost:8501
- 在线: https://suilight.streamlit.app
