## 功能规划 (优先级排序)

### P0 - 核心功能 (本周完成)

#### 1. 胶囊存储与检索

```
状态: 🔴 未开始

需求:
├── 存储: 将胶囊保存到数据库
├── 检索: 按关键词/维度/质量检索胶囊
├── 分类: 按领域分类胶囊
└── 标签: 自动标签 + 手动标签

技术方案:
├── SQLite 增加 capsules 表
├── 全文检索: SQLite FTS5
└── API: GET /api/capsules?q=关键词
```

**Milestone**: 每个讨论能自动生成胶囊并存储

#### 2. 胶囊展示页面

```
状态: 🔴 未开始

需求:
├── 胶囊列表页: 展示所有胶囊
├── 胶囊详情页: 展示胶囊完整内容
├── 胶囊搜索: 按条件筛选
└── 胶囊分享: 生成分享链接
```

**Milestone**: 用户能查看和管理所有胶囊

## 讨论: 胶囊的价值主张

### 核心问题

```
讨论过程很长 → 胶囊凝练精华

但:
1. 胶囊如何比讨论记录更有价值?
2. 胶囊如何促进知识复用?
3. 胶囊如何产生网络效应?
```

### 回答

| 问题 | 回答 |
|------|------|
| 更有价值 | 胶囊 = 核心洞见 + 质量评分 + 行动建议 |
| 促进复用 | 胶囊可搜索、可引用、可迭代 |
| 网络效应 | 胶囊之间形成引用关系，产生知识网络 |
